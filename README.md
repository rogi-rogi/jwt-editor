# JWT Editor

브라우저에서 JWT를 입력하고 헤더/페이로드를 수정한 뒤 재서명할 수 있는 정적 웹 도구입니다.

[바로가기](https://rogi-rogi.github.io/jwt-editor/)


## 개요

- JWT 디코딩(헤더/페이로드) 및 편집
- HMAC 서명(HS256/HS384/HS512) 기반 재서명
- 시크릿 키는 브라우저 내에서 JWT 재서명에만 사용

## 사용 방법

1) JWT 입력
- 상단 입력창에 JWT를 붙여넣습니다.
- 자동으로 헤더/페이로드가 표시됩니다.

2) 시크릿 키 입력
- 열쇠 버튼을 눌러 모달을 열고 시크릿 키를 입력합니다.
- 눈 아이콘으로 표시/숨김 전환이 가능합니다.

3) 헤더/페이로드 수정
- 하단 편집 영역에서 JSON을 수정합니다.
- 시크릿 키와 `alg` 값을 기준으로 JWT가 자동 재서명됩니다.

4) JWT 복사
- 복사 버튼으로 입력창의 JWT를 클립보드에 복사합니다.

## 서비스 지원 범위

- 지원 알고리즘: HS256 / HS384 / HS512
- 지원 기능: 디코딩, JSON 편집, HMAC 재서명, JWT 복사
- 비지원: RS/ES 계열(비대칭 서명), 토큰 검증 서버 API, 만료 검증

## 보안 안내

- 시크릿 키는 브라우저에서만 JWT 재서명에만 사용되며 외부 서버로 전송되지 않습니다.
- 민감한 키는 공유 환경에서 입력하지 마세요.
